# 変更履歴

Koecho の変更履歴です。GitHub Releases と version tag を照合して作成しています。英語版は [CHANGELOG.md](CHANGELOG.md) を参照してください。

リリース成果物は [GitHub Releases](https://github.com/ryotapoi/Koecho/releases) から入手できます。

## 未リリース

### 内部改善

- ディクテーション、ホットキー、スクリプト、入力パネル、音声入力のテストから実時間依存の待機を除去。
- 実行環境で利用できない Speech Analyzer テストを、空のテストとして成功させず無効として表示するよう改善。
- オーディオ入力と音声モデル検証の共有状態をテストごとに分離。
- 確定・未確定の音声入力テキスト編集と入力状態の書き込みについて、責務と経路を集約。
- 置換プレビューのツールチップ表示を分離し、専用テストを追加。
- 入力パネル controller の未使用な音声入力 delegate 転送処理を削除。
- 置換プレビューと実適用で置換エンジンを共有するよう整理。
- 置換ショートカットの既定値と optional shortcut の永続化処理を集約。
- 置換ルールの旧単数パターン形式の decode 対応を撤去。

## 1.6.3 — 2026-07-05

- 複数パターンの追加・削除時に、置換ルールの編集を安定化。
- 文字起こしの再起動と音声入力処理の信頼性を向上。
- ディクテーション開始、重複除去、表示メッセージの回帰テストを追加。
- 派生設定リストの生成をキャッシュし、性能を向上。

## 1.6.2 — 2026-07-04

- 文字起こしの再起動機能を追加。
- 音声入力の信頼性向上に向けて、オーディオデバイス選択と Speech Analyzer の判定ロジックを分離・テスト。
- タイムアウト処理と音声モデル準備処理を再利用可能な形に整理。

## 1.6.1 — 2026-07-04

- 音声入力レベル監視を改善し、専用テストを追加。
- 音声ロケールのモデルを KoechoCore に移動。
- 派生 View コレクションをキャッシュし、性能を向上。

## 1.6.0 — 2026-07-04

- ディクテーション開始処理を改善。
- 表示マッピングと音声入力テキストの重複除去に回帰テストを追加。
- クリップボード貼り付けテストの独立性を向上。

## 1.5.0 — 2026-06-25

- 入力パネルのデザインと余白を刷新。
- アプリアイコンの切り替えに対応。
- スクリプトプロンプト、ツールバーの状態表示、ダークモード表示を改善。
- README のスクリーンショットを更新。

## 1.4.2 — 2026-06-15

- 音声ロケールの変更後にメニューの言語表示を更新するよう改善。

## 1.4.1 — 2026-06-11

### 内部改善

- 入力パネルと音声入力の状態処理を整理。共通終了処理、確定処理の段階分け、挿入位置の管理、再生抑制、ロケール補正、音声モデルキャッシュを改善。
- 入力パネルと関連するプラットフォーム処理のテスト範囲を拡大。

## 1.4.0 — 2026-03-27

### 新機能

- **複数パターン置換**: 1 つの置換ルールに複数の入力パターンを登録可能にしました（例: `GitHブ`、`ギットHub` → `GitHub`）。
- **置換ルール UI の刷新**: 通常の置換モードと正規表現モードの UI を分離。

### バグ修正

- `restartTranscriber` 実行時に発生する CoreAudio IO スレッドのクラッシュを修正。
- 音声入力直後にカーソルを移動するとテキストが重複する問題を修正。

## 1.3.0 — 2026-03-22

- String Catalog による日本語ローカライズを追加。
- メニューバーから設定画面を再度開いたときに、設定画面が正しく前面に表示されるよう改善。
- メニューバーの機能しないキーボードショートカット表示を削除。
- サイドバーの「Replacement Rules」ラベルを「Replacement」／「置換」に短縮。

## 1.2.0 — 2026-03-20

- **入力パネルのツールバーを刷新**: スクリプトと置換の操作を、ボタンサイズを統一したコンパクトなツールバーに統合。
- **履歴ポップオーバーを改善**: スクロールせずに全文を読みやすくするため、「全文を表示」ポップオーバーを拡大。
- **アクセシビリティ**: アイコンのみのボタンにテキストラベルを追加し、`HistoryRow` を適切な Button に変更。

### 内部改善

- GCD（`DispatchQueue`、`DispatchWorkItem`）を Swift Concurrency（`Task`、`@MainActor`）へ移行。
- 古い API の利用を現行 API に更新。
- 大きな View を分割し、役割ごとの View struct を抽出。
- `InputPanelController` のコールバック配線を整理。

## 1.1.0 — 2026-03-18

- 音声入力エンジンに **Off** モードを追加。音声エンジンを起動せずにパネルを開けるため、ターミナルアプリで日本語を入力するときなど、キーボードだけで入力したい場合に利用できます。

## 1.0.1 — 2026-03-17

- オーディオデバイス、スクリプト設定、アクセシビリティ、音声解析、選択テキスト処理を中心に、内部コードの品質とテスト容易性を改善。

## 1.0.0 — 2026-03-12

macOS 向け軽量音声入力アプリの初回リリース。

- リアルタイム認識に対応した音声入力とキーボード編集。
- macOS Dictation と Speech framework（macOS 26 以降）を利用。
- テキスト処理用のシェルスクリプト連携。
- Claude Code headless mode 用のサンプルスクリプトを同梱。
- macOS 14.0（Sonoma）以降に対応。
